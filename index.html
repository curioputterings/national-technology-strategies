How to read this: the posture scale (banned/phased-out → absent → nascent → active → mature → frontier) is an editorial reading of each national strategy's ambition, funding, and maturity — not a quantitative index. Nuclear's red cells mark deliberate prohibitions or phase-outs (a negative choice, not a capability gap), which is why that column reads as a fault line rather than a gradient. "Frontier" is reserved for genuine global leaders in a domain. Verify specifics against the cited official sources in each country profile before citation.
